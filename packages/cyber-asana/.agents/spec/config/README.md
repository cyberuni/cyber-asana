# config

Repo-local project registry (`.agents/cyber-asana.json`, `cyber-asana config show/add/remove`) and env-var-sourced config resolution (`ASANA_TOKEN`, `ASANA_WORKSPACE` — see `src/env.ts`). CLI-only, no MCP surface. Governed by [design decision 0001](../design/decisions/0001-no-workspace-gid-in-repo-config.md) (no workspace GID in repo config).
