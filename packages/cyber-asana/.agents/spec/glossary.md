# Glossary

- **GID** — Asana's global ID for any object (task, project, workspace, etc.).
- **PAT** — Personal Access Token; the `ASANA_TOKEN` used to authenticate with the Asana API.
- **Workspace** — the top-level Asana org/account a request operates against (`ASANA_WORKSPACE`).
- **Gateway** — a domain's thin wrapper over the Asana SDK, returning unwrapped `res.data`.
- **Repo config** — the committed `.agents/cyber-asana.json` project registry (see `config/`).
- **AXI** — [Agent eXperience Interface](https://github.com/kunchenguid/axi), the 10 agent-CLI principles this project's CLI/MCP output conforms to (see `axi/`).
- **TOON** — the compact structured-output format (`--toon` / `CYBER_ASANA_MCP_FORMAT=toon`), alternative to JSON.
